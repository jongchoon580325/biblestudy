현대적인 개인 지식 관리 웹앱 UI/UX 가이드 (다크모드 기반)
1. 색상 시스템 (다크모드 기본)
주요 색상 팔레트:

배경: Deep charcoal (#1a1a1a), Midnight blue (#0f1419)
서피스: Dark gray (#2d2d2d), Card background (#252525)
액센트: Electric blue (#00d4ff), Neon green (#00ff88)
텍스트: Pure white (#ffffff), Light gray (#e0e0e0)
서브텍스트: Medium gray (#888888)

시각적 계층:

글래스모피즘 효과로 카드와 모달에 반투명 배경
서브틀한 그라데이션과 그림자로 깊이감 표현
네온 컬러 액센트로 중요한 액션 버튼 강조

2. 레이아웃 구조
메인 레이아웃:
┌─────────────────────────────────────────────────────┐
│ Header: 로고 + 검색바 + 사용자 설정                      │
├─────────────────────────────────────────────────────┤
│ Sidebar       │ Main Content Area                    │
│ - 카테고리      │ ┌─────────────────────────────────┐ │
│ - 태그 클라우드  │ │ Toolbar: 뷰모드/필터/정렬        │ │
│ - 최근 항목     │ ├─────────────────────────────────┤ │
│ - 즐겨찾기      │ │ Content Grid/Table              │ │
│ - 통계         │ │ (카드형 또는 테이블형)             │ │
│               │ └─────────────────────────────────┘ │
└─────────────────────────────────────────────────────┘
반응형 설계:

데스크톱: 3컬럼 레이아웃 (사이드바 + 메인 + 디테일 패널)
태블릿: 2컬럼 (접을 수 있는 사이드바)
모바일: 1컬럼 (햄버거 메뉴)

3. 카드 디자인 (카드형 뷰)
현대적 카드 스타일:

형태: 둥근 모서리 (border-radius: 12px)
그림자: 다층 그림자로 부유감 표현
호버 효과: 부드러운 스케일 업 + 글로우 효과
내용 구성:
┌──────────────────────────┐
│ 📄 [파일 아이콘/썸네일]     │
│ 제목 (truncated)         │
│ 카테고리 태그             │
│ #태그 #태그 #태그         │
│ 📅 2025.06.02  📊 2.3MB │
│ [액션 버튼들]             │
└──────────────────────────┘


마이크로 인터랙션:

카드 호버 시 부드러운 elevation 변화
태그 클릭 시 펄스 애니메이션
즐겨찾기 토글 시 하트 애니메이션

4. 테이블 디자인 (테이블형 뷰)
현대적 데이터 테이블:

스타일: 경계선 없는 클린한 디자인
행 구분: 서브틀한 배경색 차이 (zebra striping)
헤더: 고정 헤더 + 정렬 인디케이터
인터랙션: 행 호버 시 배경 하이라이트

5. 검색 및 필터링 UI
통합 검색바:

위치: 상단 헤더 중앙 배치
기능: 실시간 자동완성 + 검색 히스토리
비주얼: 글래스모피즘 효과 + 검색 아이콘

고급 필터 패널:

토글 방식: 슬라이드 아웃 패널
필터 옵션: 체크박스 + 슬라이더 + 날짜 피커
비주얼 피드백: 활성 필터 카운터 뱃지

6. 업로드 인터페이스
드래그 앤 드롭 존:

기본 상태: 점선 경계의 대형 드롭존
드래그 중: 네온 컬러 테두리 + 내부 글로우
업로드 중: 서큘러 프로그레스 + 파일 프리뷰

파일 업로드 모달:

레이아웃: 중앙 정렬 모달 + 블러 백드롭
진행 상황: 개별 파일별 진행률 표시
완료 상태: 체크 애니메이션 + 성공 메시지

7. 내비게이션 및 사이드바
사이드바 스타일:

접을 수 있는 디자인: 아이콘만 표시 ↔ 풀 텍스트
현재 위치 표시: 액티브 메뉴 하이라이트
카테고리 트리: 확장/축소 가능한 트리뷰

태그 클라우드:

시각화: 사용 빈도에 따른 크기 차이
인터랙션: 호버 시 관련 자료 수 표시
색상: 그라데이션 태그 색상

8. 모달 및 팝업
현대적 모달 디자인:

백드롭: 블러 효과 + 반투명 오버레이
모달 박스: 글래스모피즘 + 부드러운 애니메이션
진입/이탈: 스케일 + 페이드 트랜지션

9. 애니메이션 및 트랜지션
마이크로 애니메이션:

페이지 전환: 페이드 + 슬라이드 조합
데이터 로딩: 스켈레톤 UI + 쉬머 효과
사용자 피드백: 토스트 알림 + 바운스 애니메이션

성능 고려사항:

CSS transform과 opacity 중심의 하드웨어 가속 애니메이션
60fps 유지를 위한 적절한 duration 설정

10. 접근성 및 사용성
접근성 고려사항:

키보드 내비게이션: 탭 순서 최적화
스크린 리더: 적절한 ARIA 라벨링
색상 대비: WCAG 2.1 AA 기준 준수

사용성 향상:

컨텍스트 메뉴: 우클릭 액션 메뉴
단축키: 파워 유저를 위한 키보드 단축키
툴팁: 기능 설명을 위한 인터랙티브 툴팁
